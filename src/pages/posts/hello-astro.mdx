# Hello Astro

这是一个使用 **Astro** 构建的示例文章。Astro 是一个现代的静态站点生成器，专为构建快速、内容驱动的网站而设计。

## 为什么选择 Astro？

Astro 具有以下优势：

- **零 JavaScript 默认**: 默认情况下，Astro 生成零客户端 JavaScript
- **组件岛屿**: 可以在需要时添加交互性
- **框架无关**: 支持 React、Vue、Svelte 等多种框架
- **内置优化**: 自动优化性能

## 快速开始

使用以下命令创建新的 Astro 项目：

```bash
npm create astro@latest
```

然后按照提示完成项目创建：

```bash
cd my-astro-site
npm install
npm run dev
```

## 项目结构

Astro 项目的典型结构如下：

```
src/
├── components/
├── layouts/
├── pages/
└── styles/
```

## 创建页面

在 `src/pages/` 目录中创建 `.astro` 文件即可自动生成页面路由。

```astro
---
// 组件脚本 (在构建时运行)
const title = 'Hello World';
---

<html>
  <head>
    <title>{title}</title>
  </head>
  <body>
    <h1>{title}</h1>
  </body>
</html>
```

## 总结

Astro 为构建现代静态网站提供了一个优秀的解决方案。它的组件岛屿架构和零 JavaScript 默认设置使得网站既快速又灵活。

开始你的静态站点之旅吧！
